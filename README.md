# Keyboard Layout Switch

Automatically change keyboard layout result RUS/ENG. For example when you want to change cyrilic characters in password fields to english ones.

## Install

```

    npm i keyboard-layout-switch

```

## Usage

```
import kls from './';

kls('тЕсТ') // nTcN
```
